# SANA Video Local UI Promo Kit

This folder is the share kit for public posts, DMs, repo launch notes, and screenshots.

## Core Positioning

SANA Video Local UI is a Windows-friendly React + FastAPI workbench for running SANA-Video 2B locally. It focuses on the practical bits people need when testing video generation on consumer hardware: start images, chained segments, model profiles, low-VRAM mode, job progress, and a replay gallery.

## One-Sentence Pitch

A local React + FastAPI workbench for running SANA-Video 2B on consumer GPUs, with image starts, chained clips, low-VRAM controls, and model profiles ready for future swaps.

## Honest Scope

This is not a model release and it does not include weights. It is a practical local UI and backend wrapper around currently available SANA-Video diffusers-compatible weights. Future SANA-WM / VM support depends on what NVIDIA releases and whether the backend adapter needs to change.
