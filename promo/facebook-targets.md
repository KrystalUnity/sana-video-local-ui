# Facebook Launch Targets

Use this as a first-pass outreach list. Always check group rules before posting, and avoid posting the same copy everywhere on the same day.

## Best First Targets

### ComfyUI

URL: `https://www.facebook.com/groups/comfyui/`

Why it fits: practical local generation users, including image-to-video and hardware-setting discussions.

Angle: "I built a small React + FastAPI workbench for local SANA-Video 2B testing. It is not a ComfyUI workflow, but the local video + VRAM crowd may find the setup notes useful."

### AI Animation & AI Video

URL: `https://www.facebook.com/groups/aianimationonly/`

Why it fits: directly focused on AI video output and animation workflows.

Angle: "Open-source local video workbench, tested on a 12 GB laptop GPU, with image starts and chained segments."

### Generative AI - Midjourney, ChatGPT, Sora, Suno, DALL-E, Stable Diffusion

URL: `https://www.facebook.com/groups/generativeai.official/`

Why it fits: broad generative AI audience; good for reach if the group allows tool/project posts.

Angle: "Not a model release or hosted product; a practical open-source lab for people testing local video generation."

### AI Generated Art

URL: `https://www.facebook.com/groups/aigenart/`

Why it fits: broad AI art community; may work best if the post leads with the visual demo rather than repo details.

Angle: "Short demo of a local SANA-Video UI, plus the GitHub link for people who want to try the setup."

### ChatGPT Community

URL: `https://www.facebook.com/groups/chatgpt/`

Why it fits: large AI builder audience. Not video-specific, so use only if developer/tool posts are allowed.

Angle: "Open-source React + FastAPI tool built with local AI experimentation in mind."

## Secondary / Maybe Later

### FlowKit / Flowboard Community

URL: `https://www.facebook.com/groups/flowkit.flowboard.community`

Why it fits: AI video workflow community, but it may be more focused on FlowKit/Flowboard specifically.

Angle: "Adjacent local video generation workflow; share only if the group allows related open-source tools."

### Official Midjourney Group

URL: `https://www.facebook.com/groups/officialmidjourney`

Why it fits: AI image/video crowd, but likely less local-open-source oriented.

Angle: "Use only if there is an allowed community/share thread. Do not lead with local SANA tooling in a way that looks off-topic."

## Avoid

- Anti-generative-AI groups.
- Artist portfolio groups that prohibit AI tools.
- Groups with "no self-promo", "no links", or "no GitHub/tool drops" rules unless they have a weekly share thread.

## Posting Cadence

1. Post to personal/Facebook thread first.
2. Post to one high-fit group: ComfyUI or AI Animation & AI Video.
3. Wait for comments and questions.
4. Improve README or issues from the first feedback.
5. Post to broader groups with a refined version.

## First Group Post

Built a local UI for testing SANA-Video 2B on consumer hardware.

It is a React + FastAPI workbench with:

- text-to-video and image start frames
- chained segments for longer clips
- low-VRAM mode
- model profiles
- progress, peak VRAM, and output gallery

This is not a model release and it does not include weights. It is just a practical open-source lab for people testing local video generation.
